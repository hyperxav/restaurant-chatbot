Les Chimères Image Generator

How to use this tool:

  1. Click on each "Open Generator" button below to open the image generator in a new tab
  2. Take a screenshot of each generated image (or use your browser's save functionality)
  3. Save the images to the images folder with the appropriate filenames
  4. Use these images as placeholders in your restaurant landing page
